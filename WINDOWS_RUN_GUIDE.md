# คู่มือการรันระบบดึงข้อมูล (Sync) บน Windows 🪟

คู่มือนี้สำหรับใช้กับเครื่องคอมพิวเตอร์ตั้งโต๊ะ (Windows) เพื่อทำหน้าที่เป็น **Backend Sync Server** สำหรับดึงข้อมูลแชทและโฆษณาเข้าสู่ Supabase

---

## 1. การเตรียมเครื่อง (ครั้งแรกเท่านั้น)
หากยังไม่มีโปรแกรมเหล่านี้ในเครื่อง ให้ติดตั้งก่อนครับ:
1.  **Node.js**: [ดาวน์โหลดที่นี่](https://nodejs.org/) (แนะนำเวอร์ชัน 20 หรือ 22 LTS)
2.  **Git**: [ดาวน์โหลดที่นี่](https://git-scm.com/)

---

## 2. การดึงโค้ดล่าสุดจาก MacBook
เปิด **PowerShell** หรือ **Command Prompt** แล้วไปที่โฟลเดอร์ที่คุณต้องการเก็บงาน:

```bash
# หากยังไม่มีโฟลเดอร์โปรเจค
git clone https://github.com/Freshair129/crmapp.git
cd crmapp

# หากมีโฟลเดอร์อยู่แล้ว ให้ดึงงานล่าสุดมา
git pull origin master
```

---

## 3. การตั้งค่า Environment (.env)
สำคัญมาก! เครื่อง Windows ต้องมองเห็นฐานข้อมูลตัวเดียวกับ Mac:
1.  ตรวจสอบไฟล์ `.env` (ถ้าไม่มีให้ก๊อปปี้จาก Mac มาวาง)
2.  เช็คว่า `DATABASE_URL` ชี้ไปที่ **Supabase** ตัวจริง

---

## 4. ติดตั้ง Library และเตรียมระบบ
รันคำสั่งเหล่านี้เพื่อเตรียมตัวก่อนเริ่มงาน:

```bash
# ติดตั้งห้องสมุดต่างๆ
npm install

# สร้างตัวเชื่อมต่อฐานข้อมูล
npx prisma generate
```

---

## 5. เริ่มดึงข้อมูลแชทย้อนหลัง (90 วัน) 🚀
รันคำสั่งนี้เพื่อดึงแชทและชื่อลูกค้าทั้งหมดกลับมา:

```bash
node scripts/sync-fb-messages.mjs 90
```

*หมายเหตุ: ปล่อยทิ้งไว้จนกว่าจะเสร็จ (จะขึ้นว่า 1100/1100) ระหว่างรันคุณจะเห็นข้อมูลค่อยๆ โผล่ขึ้นมาที่ MacBook เองครับ*

---

## 6. (เพิ่มเติม) รันระบบวิเคราะห์ AI และส่ง LINE
หากต้องการให้ Windows ช่วยทำงานพื้นหลังอื่นๆ ด้วย ให้เปิดอีก Terminal หนึ่งแล้วรัน:

```bash
npm run worker
```
