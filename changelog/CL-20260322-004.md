## Summary

ปรับปรุง Employee Card UX ทั้งหมด (shimmer, opacity, FAB, My Card indicator), ทำ PermissionMatrix ให้ interactive กด-cycle-save ได้จริง, เพิ่ม `jobTitle` field แยกออกจาก `department`, และ fix ข้อมูล `facebook_url` ใน DB

---

## Changes

### Employee Card Deck
- **Linear Shimmer Effect** (Framer Motion): `motion.div` แล่น 105° x: -110%→210%, 2.2s, repeatDelay 3.5s — เฉพาะ active (front) card เท่านั้น
- **Background card opacity**: ลด bleed-through — abs=1: 0.78→0.35, abs=2: 0.56→0.12, abs>2: 0
- **FAB button**: ย้ายเข้ามาใน card (top:14, right:14, positive coords) — ลบ pseudo-concave bg-circle ออก
- **FAB gradient**: เปลี่ยนจาก `radial dark` → `linear-gradient(135deg, color0, color1)` match style avatar เป๊ะ
- **"การ์ดของคุณ" indicator**: เมื่อ `emp.email === currentUser.email` แสดง gold top-border line 2px + pill badge "◆ การ์ดของคุณ" ที่ bottom center ของ card
- **Edit button**: ย้ายออกจาก Profile card tab มาอยู่ขวาของ tab bar — เห็นได้ทุก tab (Overview/Permissions/Customers/Sales)

### PermissionMatrix
- **Interactive cells**: กดทุก cell ได้ — cycle 5 states: `true → false → 'own' → 'log' → 'request' → true`
- **Persistent**: PATCH `/api/permissions` บันทึก full permissions JSON ลง `app_config` table (Supabase)
- **Load from DB**: `GET /api/permissions` อ่าน DB overrides แล้ว merge กับ static base PERMISSIONS
- **Save button**: ปุ่มทอง "Save" โผล่เมื่อ `isDirty` — spinner ขณะ saving
- **ลบ info box "Inherited Permissions"** ออก (เคยชื่อ "Read-Only View")

### Schema & Data
- **`jobTitle` field**: เพิ่ม `Employee.jobTitle` (`job_title TEXT`) ใน Prisma schema + Supabase ALTER TABLE
- **Dept ≠ Job Title**: `department` = สังกัดทีม/encode ใน ID | `jobTitle` = ตำแหน่งงานจริง เช่น "Pastry Chef"
- **DB fix**: clear `facebook_url = 'suanranger129@gmail.com'` → NULL สำหรับ Guest + Demo employee records

### API
- `POST /api/permissions` (NEW): GET effective permissions (DB merged with static), PATCH save overrides
- `POST /api/employees`: รับ `jobTitle` ใหม่
- `PATCH /api/employees/[id]`: เพิ่ม `jobTitle` ใน allowed fields list

### Supabase
- `app_config` table: `key VARCHAR(120) PK, value JSONB, updated_at TIMESTAMPTZ`
- `employees.job_title TEXT` column added
