## Procurement PO Lifecycle (ADR-049)

### PO Status Flow
```
DRAFT → REQUEST_REVIEW → APPROVED → ORDERING → ORDERED → RECEIVING → RECEIVED → CLOSED
                       → REJECTED (ตีกลับ → แก้ไข → DRAFT)         → PARTIAL / ISSUE
                                                                      → RETURN / CREDIT NOTE
```

### Prisma Models (14)
- `Supplier` (SUP-NNN), `PurchaseOrderV2` (PO-YYYYMMDD-NNN), `POItem`
- `POApproval` (APV-), `POAcceptance` (ACC-), `POTracking` (TRK-)

### Key Features
- **BOM Calculator**: Class → Course → Recipe → Ingredient × students → shortfall → auto-create PO
- **Chef Approval**: เชฟเจ้าของ class ต้อง approve ทุก PO
- **Tracking**: ติดตามขนส่ง (PREPARING → SHIPPED → IN_TRANSIT → DELIVERED)
- **GRN → Lot**: รับของแล้ว auto-increment Ingredient.currentStock
- **Advance**: เงินทดรองจ่าย / ออกเงินส่วนตัว → ขอเบิกคืน
