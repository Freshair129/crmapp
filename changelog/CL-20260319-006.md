### Google Sheet → Auto-gen ID
- `src/lib/repositories/courseRepo.js`
  - เขียน `generateProductId({ productType, cuisineCode, packCode, subcatCode, pkgNo, pkgShortName, hours })` ใหม่
  - COURSE → `TVS-{cuisineCode}-{packCode}-{subcatCode}-{SERIAL:02d}` (serial per prefix)
  - PACKAGE → `TVS-PKG{pkgNo:02d}-{pkgShortName}-{hours}H`
  - FULL_COURSE → `TVS-FC-FULL-COURSES-{A|B}-{hours}H`
  - แก้ format เก่า `PRD-CRS-YYYY-XXX` → ตรง spec จริง
- `src/app/api/sheets/sync-master-data/route.js`
  - รองรับ columns ใหม่: `productType`, `cuisineCode`, `packCode`, `subcatCode`, `pkgNo`, `pkgShortName`
  - Auto-infer DB `category` จาก `cuisineCode` (JP → japanese_culinary, SP → specialty, ฯลฯ)
  - Row ที่ไม่มี `productId` → auto-gen จาก components
  - Comment block อธิบาย Sheet columns ครบ

## Root Cause (ID format ผิด)

`generateProductId` รุ่นแรกใช้ `PRD-CRS-YYYY-XXX` ซึ่งเป็น format ที่ Claude เขียนขึ้นเอง
ไม่ตรงกับ `id_standards.yaml` ที่กำหนดไว้ว่า `TVS-[CATEGORY]-[PACK]-[SUBCAT]-[SERIAL]`
สาเหตุ: ไม่ได้อ่าน id_standards.yaml ก่อนเขียน
