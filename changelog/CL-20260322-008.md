## 🚀 Summary
Implemented "NotebookLM-style" intelligence for conversations, providing automated daily summaries and interactive knowledge tree diagrams (Mermaid.js).

## 🛠️ Changes

### Intelligence Engine
- **Prisma Schema:** Added `ConversationIntelligence` model to store summaries, takeaways, and tree codes.
- **IntelligenceRepo:** New repository using **Gemini 2.0 Flash** to perform deep analysis of chat history.
- **Automated Mapping:** Logic to convert chat context into Mermaid.js "Knowledge Tree" structures.

### API & Integration
- **API Route:** `GET/POST /api/inbox/conversations/[id]/analysis` for manual and on-demand intelligence retrieval.
- **MCP Server v1.9.0:** Added `intelligence.get_chat_tree` tool, allowing AI Agents to fetch customer knowledge maps directly.

### Documentation
- **system_requirements.yaml:** Added Module 25 (Daily Chat Intelligence).
- **overview.md:** Updated version roadmap to v1.9.0.
