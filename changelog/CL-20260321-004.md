### Key Architecture Decisions (ADR-046)
- Receipt ผูก Order 1:1 (ไม่ใช่ 1:N — 1 order = 1 receipt)
- 3 print channels: Thermal (Web Serial), Browser (`window.print`), LINE (PNG push)
- ESC/POS commands encode ฝั่ง client — ไม่ต้อง print server
- Thai text ใช้ TIS-620 charset (code page 26)
- Receipt ID: `RCP-YYYYMMDD-XXX` (daily serial reset)
